## Kanban Board Mini-Project README

# 📋 Kanban Board - Next.js

A high-performance, responsive Kanban board built with **Next.js**, **Tailwind CSS**. This project demonstrates modern frontend patterns for managing task workflows, handling complex drag-and-drop interactions, and maintaining persistent state.



### 🚀 Key Features

*   **Drag-and-Drop Workflow:** Seamlessly move tasks between columns (To Do, In Progress, Done) with smooth animations.
*   **Task Management:** Create, edit, and delete tasks with instant UI updates.
*   **Persistent State:** Tasks are saved to postgress to ensure data persists across sessions.
*   **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
*   **Accessible UI:** Built with accessibility in mind, ensuring keyboard navigation support for drag-and-drop actions.
